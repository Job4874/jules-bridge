# VRL Language Support

Local VS Code extension scaffold for Vector Remap Language files.

## Features

- `.vrl` language registration
- TextMate syntax highlighting
- Snippets
- Builtin completions from `src/data/vrl-builtins.json`
- Hover documentation from the same builtin data
- Lightweight diagnostics for unmatched brackets and unknown builtin calls

## Development

```powershell
npm install
npm run compile
npm run watch
```

Press F5 in VS Code from this folder to launch an Extension Development Host.
